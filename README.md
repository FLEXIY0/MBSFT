CLI manager for Minecraft Beta 1.7.3 servers on Termux (Android).

## Install

Run in Termux:

```bash
pkg install -y wget && wget https://raw.githubusercontent.com/FLEXIY0/MBSFT/main/install.sh -O mbsft && chmod +x mbsft && ./mbsft
```

## Update

The script auto-updates on launch.

## Use

- **mbsft** - open menu

## Features

- Multiple servers
- RAM/Port config
- Auto-backups (via services)
- SSH management
- Servers: Poseidon, Reindev, FoxLoader
